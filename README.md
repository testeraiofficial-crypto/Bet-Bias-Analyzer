# 📉 GR Analyzer - Behavioral Finance Simulator

Simulator interaktif untuk memvisualisasikan bagaimana **Gambler's Fallacy ($G$)** mempercepat kehabisan modal berdasarkan volume taruhan dan RTP.

### Cara Akses via HP
1. Upload semua file ke repositori GitHub.
2. Masuk ke **Settings** > **Pages**.
3. Pilih branch `main`, lalu Save.
4. Tunggu 1 menit, dashboard kamu akan live di: `https://username.github.io/nama-repo/`

### Analisis Utama
Aplikasi ini menunjukkan bahwa strategi sesat seperti mengejar kekalahan ($G > 0$) akan membuat kurva saldo menurun jauh lebih tajam daripada ekspektasi statistik murni.
